# OpenClaw 桌面版

基于 StanleyChanH/openclaw-offline-package 封装的桌面应用程序。

## 使用方法

### 1. 准备离线包

从 GitHub 下载最新的离线安装包：
https://github.com/baimeng1758354676/openclaw-offline-package/releases

下载 `openclaw-offline-package-v*.zip` 并解压。

### 2. 复制文件

将解压后的所有文件（包括 `01_首次配置.bat`、`02_启动服务.bat`、`nodejs`、`node_modules` 等）复制到本项目的 `openclaw/` 目录下。

### 3. 安装依赖

```bash
npm install
```

### 4. 运行开发版本

```bash
npm start
```

### 5. 打包成 exe

```bash
npm run dist
```

打包完成后，`dist/` 目录下会生成 `OpenClaw Setup.exe` 安装程序。

## 功能

- 🦞 一键启动 OpenClaw 服务
- ⏹ 一键停止服务
- ⚙ 首次配置（配置 AI 模型等）
- 🌐 自动打开浏览器界面
- 📊 实时显示运行状态

## 注意事项

- 首次使用需要先点击「首次配置」进行 AI 模型配置
- 配置过程需要联网（OAuth 认证）
- 启动后会自动在 http://127.0.0.1:18789 运行
